VCard QR Code Generator

Overview

This module helps to generate a unique VCard QR code for Employees.

Features

  • Generate QR code for employees.
  • QR codes can be accessible via mobile devices.
  • QR codes can contain basic employee information such as name, position, and contact details.This information helps in quick identification.
